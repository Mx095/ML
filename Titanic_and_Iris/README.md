# First ML project.


## Titanic Survival Predictor
Built a machine learning model to predict whether a Titanic passenger survived or not.

- **Dataset:** Real Titanic passenger data (891 passengers)
- **Model:** Decision Tree Classifier
- **Accuracy:** 74.8%
- **Tools:** Python, Pandas, Scikit-learn, Jupyter Notebook

### What it does
Takes a passenger's details (age, gender, ticket class, fare) and predicts whether they survived the Titanic disaster.
