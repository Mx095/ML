# v2_CreditRiskEvaluator

## Purpose :
The Same CreditRiskEvaluator Model that extends into concepts like precision, recall, F1 score and support.  These are few metrics that ensures the model's precision and delivers accurate results of the model.  turning this project from a TOY project to an actual production one.

## Algorithms used :
Used RandomForestClassifier, Classification_Report and Confusion matrix 

## Results :
**Row - 0** : Borrowers who repaid
**Row - 1** : Borrowers who defaulted

**Precision** - 0 : 0.93, 1 : 0.96

**Recall** - 0 : 0.99, 1 : 0.73

**Support** - 5105 repaid, 1412 defaulted.

**Overall Accuracy** : 93.2%

## Tools used :
Matplotlib, Seaborn, Python, Pandas, Scikit-learn, Jupyter Notebook.
