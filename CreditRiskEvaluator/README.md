# CreditRiskEvaluator

A machine learning model that predicts whether a loan applicant will default or repay.

## Details
- Dataset: Real world credit risk data (32,000+ borrowers)
- Model: Random Forest Classifier
- Accuracy: 93.3%
- Tools: Python, Pandas, Scikit-learn, Jupyter Notebook

## Key Findings
The most important factors in predicting loan default:
1. Loan amount as percentage of income
2. Person's income
3. Loan grade
4. Interest rate

## What it does
Takes a borrower's details and predicts whether they will repay or default on their loan.
